lcmap.see.config

LCMAP SEE configuration

The LCMAP REST server will extract configuration values from the following, in order of highest precedence to lowest:

  • System evnironment variables
  • The ‘LCMAP Server’ section of the config/INI file in ~/.usgs/lcmap.ini (same file as used by the LCMAP client libraries)
  • The values nested under the project.clj file’s :env key
